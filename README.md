## Colors

### NPM

```bash
# npm
npm i @pebblekit/colors
```

```css
/* main.css */
@import '@pebblekit/colors';
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/@pebblekit/colors/dist/colors.min.css">
```

### Usage

```html
<h1 class="heading">Hello World!</h1>
```

```css
.heading {
  color: var(--color-red);
}
```
